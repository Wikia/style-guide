Welcome

This is Wikia's first draft of a style guide and represents a work-in-progress, with the goal of changing in smaller increments over time, as styles are tested with real world use cases.

The document is primarily aimed at international designers and developers at this time in order to drive greater consistency across design and front-end development, but it is available to anyone interested.

Upcoming Style Guide Projects

The style guide will be evolved in the context of larger projects, in order to test use cases in a live environment with the broader Wikia community.

Known upcoming projects include:

  • Abstraction of an initial user eperience pattern library
  • Adjustments for mobile and tablet web
  • Inclusion of style guide for Wikia native apps across iOS and Android
